HTML list are used to specify Lists of information. All lists may contain one or more elements. There are three types of HTML lists
  • ordered list ol
    1. type 1- the list item will be numbered with number (default)
    2. type A- the list item will be numbered with uppercase case letter
    3. type a- the list item will be numbered with lowercase case letter
    4. type I- the list item will be numbered with uppercase roman numbers
    5. type i- the list item will be numbered with lowercase roman numbers
  • Unorder list ul
  • Description list dl
    • dl
    -a description list
    dt
    -a term (name) in a description list
    dd
    -the term in a description list
