HTML lists are used to specity lists of information. All lists may contain one or more elements. There are thee types of lists:

  • Unoredered list
  • Ordered list
    1. type 1: the list items will be numbered with numbers (default)
    2. type A: the list items will be numbered with uppercase letter
    3. type a: the list items will be numbered with lowercase letter
    4. type I: the list items will be numbered with uppercase roman letter
    5. type i: the list items will be numbered with lowercase roman letter
  • Description list
    • ul element
    -defines the unoldered list.
    ol element
    -defines the ordered list.
    li element
    -defines the list item.
    dl element
    -defines the description list.
    dt element
    -defines a term (name) in a description list.
    dd element
    -defines the term in the description list.
